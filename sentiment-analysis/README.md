# Product Review Sentiment Analysis

A web-based tool that analyzes the sentiment of product reviews and determines if they are positive, negative, or neutral.

## Features

- Real-time sentiment analysis
- Visual sentiment indicators (emojis)
- Identifies positive and negative words
- Confidence score display
- Percentage breakdown (positive/neutral/negative)
- Example reviews to test
- Clean, responsive interface

## Usage

Simply open `index.html` in your browser and:
1. Enter a product review in the text area
2. Click "Analyze Sentiment"
3. View the results with detailed breakdown

Or click one of the example buttons to see how it works!

## How It Works

The analyzer uses a keyword-based approach:
- Scans the review for positive and negative words
- Calculates sentiment scores based on word frequency
- Displays confidence levels and detailed breakdowns

## No API Required

This tool works completely offline with no API keys needed!
